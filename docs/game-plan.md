# 구구단 타임어택 기획 문서

## 목표

초등학생도 바로 시작할 수 있는 1인용 구구단 게임을 만든다.

게임의 핵심은 아래와 같다.

- 1분 동안 최대한 많은 문제를 푼다
- 문제는 `2단-9단` 범위에서 랜덤으로 나온다
- 답은 큰 숫자패드로 입력한다
- 오답은 바로 처리하고 다음 문제로 넘어간다
- 마지막에 결과를 숫자로 쉽게 보여준다

## 화면 흐름

### 1. 시작 화면

- 게임 이름
- 1분 규칙 설명
- 문제 범위와 오답 처리 방식
- `1분 시작` 버튼

### 2. 플레이 화면

- 현재 문제
- 남은 시간
- 입력 중인 답
- 시도, 정답, 오답 기록
- 전화기 느낌 숫자패드

### 3. 결과 화면

- 최종 정답 수
- 총 시도 수
- 오답 수
- 정답률
- `다시 하기` 버튼

## 게임 규칙

- 제한 시간은 `60초` 고정이다.
- 문제는 두 수를 랜덤으로 뽑아 `a x b` 형태로 보여준다.
- 직전 문제와 완전히 같은 문제는 다시 뽑는다.
- 답은 두 자리까지 입력할 수 있다.
- `확인`을 누르면 즉시 정답 여부를 판정한다.
- 오답은 멈추지 않고 바로 다음 문제로 넘어간다.
- 시간이 끝나면 즉시 입력을 막고 결과 화면으로 넘어간다.

## UX 원칙

- 문장은 짧고 단순하게 쓴다.
- 중요한 숫자는 크게 보이게 한다.
- 모바일에서 손가락으로 누르기 쉬운 크기의 버튼을 사용한다.
- 숫자패드는 옛날 전화기 버튼처럼 둥글고 분명한 눌림 느낌을 준다.
