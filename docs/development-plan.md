# 개발 계획

## 1단계

- 정적 파일 구조 구성
- 시작, 플레이, 결과 화면 설계
- GitHub Pages 배포 워크플로 추가

## 2단계

- 60초 타이머 구현
- 랜덤 구구단 문제 생성
- 숫자패드 입력과 제출 로직 구현

## 3단계

- 분수 덧셈·뺄셈 모드 추가
- 초급, 중급, 고급 문제 규칙 추가
- 분자/분모 입력 UI 추가
- 모바일 화면 다듬기
- 정답/오답 피드백 정리
- README와 운영 문서 정리

## 검수 포인트

- 숫자패드 버튼이 모바일에서 충분히 큰가
- 분자/분모 입력칸이 모바일에서도 눌리기 쉬운가
- 시도, 정답, 오답 수가 정확하게 맞는가
- 시간이 끝났을 때 입력이 더 이상 되지 않는가
